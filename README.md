## Docker

```sh
# ビルド
docker-compose build

# WEBレイアウト確認
docker-compose up -d nginx
# access to http://localhost:8000

# goApp/TodoApp
docker-compose up -d todo_app
# access to http://localhost:8000/gin/todoapp/

# pgAdmin
docker-compose up -d pgadmin
# access to http://localhost:8001
```
